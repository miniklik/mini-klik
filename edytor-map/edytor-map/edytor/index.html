🌐 Dodaj panoramę 360°

📷 Jak dodać zdjęcie sferyczne? ▼
  1. Zrób zdjęcie 360° (np. aplikacją Google Street View) — format sferyczny / equirectangular, proporcje 2:1.
  2. Zmień nazwę pliku na: rynek-360.jpg
  3. Wgraj plik do folderu img/ na serwerze (obok pliku index.html).
  4. Gotowe — pinezka 🌐 pokaże panoramę po kliknięciu płatka.

Dozwolone formaty: .jpg, .png, .webp. W polu wpisz samą nazwę (bez -360.jpg).
