# Studio — Edytor stron HTML

**Autorski edytor wizualny do tworzenia stron internetowych, prezentacji, kartek, życzeń i grafik.**

Działa w przeglądarce, bez instalacji. Jeden plik HTML — pełnia możliwości.

🌐 **Strona projektu:** [mini-klik.pl](https://mini-klik.pl)

---

## ✨ Co potrafi

- 📐 Dowolne rozmiary sceny (100–5000 px)
- 📝 Pole tekstowe z formatowaniem (jak w Wordzie)
- 🎨 Kształty: kwadrat, prostokąt, koło, gwiazda
- 🖌 Pędzel z dowolnym kolorem i grubością
- 😀 Biblioteka 1200+ emotek z kategoriami i wyszukiwarką
- 🌈 Tła, gradienty, kolor obszaru wokół sceny
- 🎬 Animacje obiektów + przejścia między poziomami
- 🔍 Zoom kółkiem myszy
- 📚 Biblioteka życzeń (wczytywanie gotowych szablonów)
- 💾 Zapis projektu, eksport do HTML
- 📱 Responsywny eksport (tryb pionowy = scrollowanie strony WWW)
